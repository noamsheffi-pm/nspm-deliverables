Pattern

Each agent is a named persona over scheduler + skills + state files. Not a new runtime, not magic. Identity gives them a voice, scheduler gives them autonomy, state files give them memory across runs. New agents follow the same shape: pick a Mario character, pin a job nobody currently owns, wire a cron, write a memory entry.
